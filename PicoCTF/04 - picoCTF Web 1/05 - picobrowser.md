# Notas adicionales
Un **agente de usuario** (o _user agent_) es una cadena de texto que un cliente, como un navegador web, envía al servidor web durante una solicitud HTTP. Esta información identifica al dispositivo y al software que está accediendo a una página, incluyendo detalles como el nombre del navegador, su versión, el sistema operativo, el tipo de dispositivo (escritorio, móvil, tablet) y el idioma configurado.
