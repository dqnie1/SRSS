# Notas adicionales
La **ofuscación** es el proceso de hacer que algo sea difícil de entender, especialmente con el fin de ocultar su significado, propósito o funcionamiento.  En el ámbito de la **informática y la ciberseguridad**, se refiere a transformar el código fuente, el código intermedio o el código máquina de un programa de forma que mantenga su funcionalidad original, pero sea mucho más difícil de comprender, analizar o modificar por terceros.  Esta técnica se utiliza principalmente para **proteger la propiedad intelectual**, **evitar la ingeniería inversa**, **ocultar algoritmos sensibles** o **dificultar la detección de malware** por sistemas antivirus.
